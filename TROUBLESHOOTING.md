# 🔧 Troubleshooting & FAQ\n\n## 🚨 Common Issues\n\n### 1. Port 3000 Already in Use\n\n**Problem:** `Error: listen EADDRINUSE: address already in use :::3000`\n\n**Solution:**\n\n**Windows:**\n```bash\nnetstat -ano | findstr :3000\ntaskkill /PID <PID> /F\n```\n\n**macOS/Linux:**\n```bash\nlsof -i :3000\nkill -9 <PID>\n```\n\n**Or change port in .env:**\n```env\nPORT=3001\n```\n\n---\n\n### 2. Database Connection Failed\n\n**Problem:** `Error: connect ECONNREFUSED 127.0.0.1:3306`\n\n**Solution:**\n\n**Start MySQL:**\n\n**Windows:**\n```bash\nnet start MySQL80\n```\n\n**macOS:**\n```bash\nbrew services start mysql\n```\n\n**Linux:**\n```bash\nsudo systemctl start mysql\n```\n\n**Check credentials in .env:**\n```env\nDB_HOST=localhost\nDB_USER=root\nDB_PASSWORD=your_password\nDB_NAME=xiangqi_db\n```\n\n---\n\n### 3. Unknown Database Error\n\n**Problem:** `Error: Unknown database 'xiangqi_db'`\n\n**Solution:**\n\nServer sẽ tự tạo database, nhưng nếu không:\n\n```bash\n# Login MySQL\nmysql -u root -p\n\n# Tạo database\nCREATE DATABASE xiangqi_db;\nEXIT;\n```\n\n---\n\n### 4. npm Dependencies Error\n\n**Problem:** `Module not found: 'express'`\n\n**Solution:**\n```bash\ncd backend\nnpm install\n```\n\n**Nếu vẫn lỗi:**\n```bash\nrm -rf node_modules package-lock.json\nnpm install\n```\n\n---\n\n### 5. CORS Error in Browser\n\n**Problem:** `Access to XMLHttpRequest blocked by CORS policy`\n\n**Solution:**\n\nKiểm tra .env:\n```env\nFRONTEND_URL=http://localhost:3000\n```\n\nHoặc cài đặt CORS middleware trong server.js:\n```javascript\napp.use(cors({\n  origin: 'http://localhost:3000',\n  credentials: true\n}));\n```\n\n---\n\n### 6. JWT Token Invalid/Expired\n\n**Problem:** `Invalid token` hoặc token không được nhận\n\n**Solution:**\n\n**Check localStorage:**\n```javascript\n// Browser console\nconsole.log(localStorage.getItem('token'));\n```\n\n**Clear localStorage:**\n```javascript\nlocalStorage.clear();\n```\n\n**Check JWT Secret:**\nĐảm bảo `JWT_SECRET` trong .env giống nhau ở backend\n\n**Regenerate token:**\n- Đăng xuất\n- Xóa localStorage\n- Đăng nhập lại\n\n---\n\n### 7. Password Hashing Issues\n\n**Problem:** Login luôn thất bại mặc dù password đúng\n\n**Solution:**\n\nĐảm bảo bcrypt được cài:\n```bash\nnpm install bcrypt\n```\n\n**Check password length:**\n- Password tối thiểu 6 ký tự\n- Không có khoảng trắng đầu/cuối\n\n---\n\n### 8. Brightness/Volume Not Saving\n\n**Problem:** Slider hoạt động nhưng không lưu vào database\n\n**Solution:**\n\n**Check token:**\n```javascript\nconst token = localStorage.getItem('token');\nconsole.log(token); // Should have value\n```\n\n**Check API response:**\n```javascript\nfetch('http://localhost:3000/api/auth/brightness', {\n  method: 'PUT',\n  headers: {\n    'Authorization': `Bearer ${token}`,\n    'Content-Type': 'application/json'\n  },\n  body: JSON.stringify({ brightness: 50 })\n}).then(r => r.json()).then(console.log);\n```\n\n**Check server logs:**\nXem console của server, có error message không?\n\n---\n\n### 9. Dashboard Shows Wrong User Data\n\n**Problem:** Dashboard hiển thị thông tin người dùng khác\n\n**Solution:**\n\n**Clear cache & localStorage:**\n```javascript\nlocalStorage.clear();\nwindow.location.reload();\n```\n\n**Check API response:**\n```javascript\nconst token = localStorage.getItem('token');\nfetch('http://localhost:3000/api/auth/user', {\n  headers: { 'Authorization': `Bearer ${token}` }\n}).then(r => r.json()).then(console.log);\n```\n\n---\n\n### 10. Logout Not Working\n\n**Problem:** Logout button không tự động redirect\n\n**Solution:**\n\n**Check browser console:**\nAnyway error messages?\n\n**Manual logout:**\n```javascript\nlocalStorage.clear();\nwindow.location.href = 'login.html';\n```\n\n**Check logout button ID:**\nĐảm bảo element có `id=\"logoutBtn\"`\n\n---\n\n## 🔍 Debugging Tips\n\n### Enable Debug Logging\n\n**Backend (.env):**\n```env\nLOG_LEVEL=debug\n```\n\n**Frontend (Browser Console):**\n```javascript\n// Check token\nconsole.log('Token:', localStorage.getItem('token'));\n\n// Check user data\nconsole.log('User:', JSON.parse(localStorage.getItem('user')));\n\n// Monitor API calls\nwindow.addEventListener('fetch', e => console.log(e));\n```\n\n### Check Network Requests\n\n1.
